## how to start

1. minesweeper 사이트 접속 (https://minesweeper.online/ko/)
2. 추측없이모드 -> 매우어려움 클릭(기본값)
3. 지뢰칸 전체가 화면안에 보이도록 위치시키고 코드 전부 실행
4. WIDTH,HEIGHT,remain_mine 변수 조정하면서 다른 난이도도 클리어 가능
